# EnvConfig Manager（环境配置管理器）需求文档

## 1. 项目核心定位

- **产品名称（暂定）**：EnvConfig Manager（环境配置管理器）
- **核心目标**：一款跨平台、可视化的系统配置管理工具，聚焦解决开发者手动编辑 shell 配置文件（.zshrc/.bashrc）、系统环境变量的低效、易出错问题，提供「编辑 - 校验 - 生效 - 备份 - 切换」全流程能力。
- **目标用户**：
  - 核心：开发 / 运维工程师（新手→中阶）
  - 次要：数据分析、设计等需配置开发环境的非技术岗人群
- **产品调性**：轻量、易用、稳定（无配置丢失风险）、跨平台（Windows/macOS/Linux 统一体验）

---

## 2. 技术选型

| 层级 | 推荐方案 | 备选方案 | 说明 |
| --- | --- | --- | --- |
| 桌面框架 | Tauri 2.x | Electron | Tauri 体积更小（约 10-20MB）、性能更好、安全性更高；Electron 生态更成熟，三方库丰富 |
| 前端框架 | Vue 3 + TypeScript | React + TypeScript | Vue 上手快、文档友好，契合目标用户（新手→中阶）的产品定位 |
| 代码编辑器 | CodeMirror 6 | Monaco Editor | CodeMirror 体积更小、可按需加载，适合轻量定位；Monaco 功能强大但体积较大（约 4MB） |
| 数据存储 | JSON 文件（本地） | SQLite | 配置数据结构简单、数据量小，JSON 文件方案足够，无需引入数据库依赖 |
| 跨平台命令 | Tauri Rust 后端 + Shell 命令 | Node.js child_process | Tauri 方案通过 Rust sidecar 执行系统命令，安全性更高 |

## 3. 功能需求（分优先级）

### P0（核心）

#### 配置文件管理

| 功能点 | 详细说明 |
| --- | --- |
| 1. 自动识别配置文件 | 启动时扫描 macOS/Linux 的 `~/.zshrc`、`~/.bashrc`、`~/.bash_profile`；Windows 的 PowerShell 配置文件（`$PROFILE`），列展示文件路径 + 最后修改时间 |
| 2. 可视化编辑 | 内置编辑器支持语法高亮（shell/PowerShell）、行号、自动缩进，支持撤销 / 重做 |
| 3. 一键生效 | 修改后点击「生效」，自动执行 source 配置文件（类 Unix）/ 刷新环境变量（Windows），无需手动输命令 / 重启终端 |

#### 环境变量管理

| 功能点 | 详细说明 |
| --- | --- |
| 1. 多维度展示 | 区分「用户级 / 系统级」环境变量，按名称、值、生效范围、是否自定义分类展示 |
| 2. 增删改查 | - 新增：输入变量名、值、生效范围，自动校验格式（如无重复、无非法字符）<br>- 修改 / 删除：系统级需提示管理员权限<br>- 查询：支持关键词搜索 |
| 3. 跨平台兼容 | Windows：调用 `[Environment]::SetEnvironmentVariable()` 写入；macOS/Linux：写入配置文件 + 临时 export |

#### 备份与回滚

| 功能点 | 详细说明 |
| --- | --- |
| 1. 自动备份 | 修改配置文件 / 环境变量前，自动生成快照（文件名 + 时间戳），存储在本地指定目录（默认 `~/.envconfig/backups/`） |
| 2. 手动备份 / 回滚 | 支持手动创建备份（可备注用途）、选择历史快照一键回滚，回滚前二次确认 |
| 3. 备份管理 | 展示所有备份快照（时间、备注、文件大小），支持删除过期备份 |
| 4. 备份容量策略 | 默认保留最近 50 个快照，磁盘占用上限 100MB；超出时自动清理最旧备份，清理前弹窗通知用户 |

#### 权限提升

| 功能点 | 详细说明 |
| --- | --- |
| 1. 系统级操作鉴权 | 修改系统级环境变量 / 配置文件时，触发系统原生鉴权弹窗：macOS 通过 `osascript` 调起授权对话框；Linux 通过 `pkexec`（PolicyKit）；Windows 通过 UAC 提权 |
| 2. 提权失败降级处理 | 用户取消授权或鉴权失败时：取消本次操作、保持原配置不变，给出明确提示（如「需要管理员权限才能修改系统环境变量，本次操作已取消」） |

### P1（进阶）

#### 配置校验

| 功能点 | 详细说明 |
| --- | --- |
| 1. 语法校验 | 保存时调用 `bash -n` / `zsh -n` 对配置文件做语法检查，捕获括号/引号不匹配、语法错误等问题，给出红色提示 + 行号定位 + 修复建议 |
| 2. 路径存在性检测 | 对 PATH 类环境变量中的路径，在保存时批量检测是否存在，不存在的路径标黄警告（非阻断） |
| 3. 冲突检测 | 检测重复的环境变量名、配置文件中重复的 alias 定义，提示合并 / 删除 |

#### 多配置切换

| 功能点 | 详细说明 |
| --- | --- |
| 1. 配置集管理 | 支持创建「开发 / 测试 / 生产」等配置集（包含配置文件片段 + 环境变量），一键切换 |
| 2. 切换冲突处理 | 切换配置集时，若当前配置有未保存的修改，弹窗提示「保存 / 丢弃 / 取消」；支持 diff 对比当前配置与目标配置集的差异 |
| 3. 配置集导出 / 导入 | 导出为 JSON 文件，支持跨设备导入 |

### P2（增值）

#### 模板库

| 功能点 | 详细说明 |
| --- | --- |
| 1. 内置模板 | 提供常用模板：Node/Python/Java 环境变量、前端 / 后端常用 shell alias |
| 2. 自定义模板 | 支持将当前配置保存为自定义模板，备注用途 |

### P3（体验）

#### 基础设置

| 功能点 | 详细说明 |
| --- | --- |
| 1. 主题切换 | 浅色 / 深色主题、编辑器字体大小调整 |
| 2. 开机自启 | 可选是否开机自启（Windows 注册表 / macOS 启动项） |
| 3. 日志查看 | 展示操作日志（如「2026-03-18 10:00 修改 .zshrc 并生效」），支持清空 |
| 4. 自动更新 | 启动时检查新版本，提示用户下载更新（Tauri 内置 updater / Electron 使用 electron-updater） |

#### 云同步

> 注：云同步涉及后端服务、账号体系、数据加密传输，复杂度较高，建议作为独立里程碑单独规划。

| 功能点 | 详细说明 |
| --- | --- |
| 1. 账号绑定 | 支持 GitHub/Gitee 账号登录（无需自建账号） |
| 2. 配置同步 | 备份快照 / 配置集同步到云端，多设备同步 |

## 4. 非功能需求

| 类型 | 具体要求 |
| --- | --- |
| 兼容性 | 支持 Windows 10+ / macOS 12+ / Ubuntu 20.04+；支持 zsh/bash/PowerShell/CMD |
| 性能 | 启动时间 ≤ 3s；编辑 / 生效操作 ≤ 1s；大配置文件（1000 行+）编辑无卡顿 |
| 安全性 | 1. 修改系统级配置需触发系统原生鉴权弹窗<br>2. 备份文件本地加密存储，密钥通过系统密钥链管理（macOS Keychain / Windows Credential Manager / Linux libsecret）<br>3. 无联网时可正常使用核心功能 |
| 易用性 | 1. 新手引导（首次启动展示核心功能教程）<br>2. 操作错误时给出明确提示（如「修改系统环境变量需管理员权限，本次操作已取消」） |
| 打包体积 | 安装包体积 ≤ 100MB（Electron 方案）/ ≤ 50MB（Tauri 方案） |
| 数据迁移与卸载 | 1. 卸载时弹窗询问是否保留备份数据（默认保留）<br>2. 跨版本升级时自动兼容历史配置数据，若格式变更则启动时自动迁移 |
